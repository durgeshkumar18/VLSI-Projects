# 8-Bit Arithmetic Logic Unit (ALU) - VLSI Design

**Internship:** CODTECH IT SOLUTIONS | **Domain:** VLSI Design | **Duration:** 6 Weeks  
**Intern ID:** CT06WK173 | **Mentor:** Neela Santosh Kumar

## 📌 Project Overview
Designed, implemented, and verified an 8-bit Arithmetic Logic Unit using Verilog HDL as part of CODTECH VLSI Internship. The ALU performs core arithmetic and logical operations essential for CPU architecture. Gained hands-on experience in complete RTL design flow from coding to synthesis.

## 🛠️ Tech Stack & Tools
**Languages:** Verilog HDL, Testbench Development  
**Tools:** Xilinx Vivado Design Suite, Vivado Simulator  
**Core Concepts:** RTL Design, Digital Logic, Combinational Circuits, Functional Verification, FSM

## ⚡ Key Features & Responsibilities
- **RTL Development:** Coded 8-bit ALU module supporting Addition, Subtraction, AND, OR, XOR, NOT, Logical & Arithmetic Shift operations using Verilog HDL.
- **Testbench & Debugging:** Developed a comprehensive self-checking testbench `alu_tb.v` for functional verification. Performed debugging and troubleshooting using Vivado Simulator by analyzing waveform outputs to fix logical errors and timing issues.
- **Simulation & Verification:** Executed behavioral simulation to verify functionality across all operation codes. Ensured 100% functional coverage for implemented opcodes.
- **Synthesis & Analysis:** Synthesized RTL design on Xilinx Vivado. Analyzed RTL schematics, timing summary, and device utilization reports for FPGA implementation.
- **Documentation:** Prepared detailed design documentation and simulation reports as per internship guidelines.

## 📊 Key Learnings & Outcomes
1. **Hands-on RTL Debugging:** Gained practical experience in troubleshooting hardware logic issues using simulation waveforms, a core skill for verification engineers.
2. **End-to-End Design Flow:** Understood complete VLSI design cycle from specification, RTL coding, testbench creation, simulation, to synthesis.
3. **Industry Tools:** Acquired proficiency in Xilinx Vivado, the industry-standard FPGA design tool used by companies like HARMAN and Intel.
4. **Problem Solving:** Resolved critical bugs related to carry propagation and opcode decoding through systematic debugging.
